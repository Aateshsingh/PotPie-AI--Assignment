## ğŸ’¡ How It Works

### User Flow
1. Student opens the app
2. Pastes code into textarea
3. Selects programming language
4. Clicks "Review Code"
5. AI agent analyzes code
6. Results shown with:
   - Review summary
   - Specific suggestions
   - Severity level

### Backend Flow
1. Validate input
2. Create prompt with code + language
3. Send to Pydantic AI agent
4. Agent calls OpenRouter API
5. LLM provides structured response
6. Parse and return to frontend

### AI Agent Details
- **Model**: Llama 2 7B (free tier)
- **System Prompt**: Expert code reviewer persona
- **Input**: Code snippet + programming language
- **Output**: Review, suggestions, severity level
- **Response Time**: 3-5 seconds (including API latency)
